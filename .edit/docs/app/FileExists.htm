FileExists

The FileExists method checks if a file exists in internal or exernal file storage.

 exists = app.FileExists( fileName );

See also FolderExists


Example

function OnStart()
{
  exists = app.FileExists( "/sdcard/wftest.txt" );
   app.ShowPopup( exists );
}
